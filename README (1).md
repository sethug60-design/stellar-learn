# ⚗️ ConceptLab

> **Ask anything. Learn at your level.**

ConceptLab is an AI-powered learning web app that explains any concept in a way that matches *your* level — from Middle School all the way to Postgraduate. Built as a first serious GitHub project by a 10th-grade student.

---

## ✨ Features

- 🎓 **7 Learning Levels** — Middle School, High School, Beginner, Intermediate, Advanced, UG, PG
- 📖 **Smart Explanation** — Simple breakdown suited to your level
- 🪜 **Step-by-Step Solution** — For problems that need walkthrough
- 🌍 **Real-Life Analogy** — Connects the concept to everyday life
- 🎯 **Why It Matters** — So you never wonder "when will I use this?"
- 📝 **Notes** — Clean bullet-point summary to save or revise
- 🃏 **Flashcards** — Click to flip! Great for quick revision
- 🧪 **Quiz** — Test yourself with instant feedback
- 💬 **Chat** — Ask follow-up questions (AI-powered in V2)

---

## 🚀 How to Run

No installs. No server. Just 3 files.

1. Download or clone this repository
2. Make sure all 3 files are in the **same folder**:
   ```
   ConceptLab/
   ├── index.html
   ├── style.css
   └── script.js
   ```
3. Double-click `index.html` — it opens in your browser

That's it! ✅
